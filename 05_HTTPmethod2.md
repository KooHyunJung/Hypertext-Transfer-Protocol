## 01 클라이언트에서 서버로 데이터 전송 ##
- 클라이언트에서 서버로 어떻게 데이터를 전송할까?
- HTTP API 설계 예시

<details>
  <summary>
    <h3> 클라이언트에서 서버로 데이터 전송 </h3>
  </summary>

1) 쿼리 파라미터를 통한 데이터 전송
- URI 끝에 쿼리 파라미터를 넣어서 전송하는 방법 
    - GET
    - 주로 정렬 필터(검색어)
2) HTTP [메시지 바디]를 통한 데이터 전송
    - POST, PUT, PATCH 주로 사용
    - 언제? -> 회원 가입, 상품 주문, 리소스 등록, 리소스 변경 등등..
